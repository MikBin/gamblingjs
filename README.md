# GamblingJS

A library for evaluating various poker and casino games
# Usage (typescript)
```bash
npm install gamblingjs
```
```javascript
import * as GAMBLINGJS from "gamblingjs"
```

### Tech
tech used
* [Typescript] 
* [Rollup] 
* [Jest] 

### Installation
clone repo then:

```sh
$ npm install 
$ npm run build
```
