# Composables

This directory contains reusable Vue Composition API logic.

## Planned Composables

Refer to [Technical Specification](../TECHNICAL_SPECIFICATION.md) for implementation details.

- **useCardSelection.ts**: Manages card selection state and validation.
- **usePokerEvaluator.ts**: Integrates with `gamblingjs` for hand evaluation.
- **useMonteCarlo.ts**: Manages simulation web workers and state.
