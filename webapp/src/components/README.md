# Components

This directory contains the Vue.js components for the application.

## Planned Components

Refer to [Technical Specification](../TECHNICAL_SPECIFICATION.md) for detailed props and behavior.

- **Card.vue**: Displays a single playing card.
- **CardSelector.vue**: Interface for selecting cards from a deck.
- **HandDisplay.vue**: Shows player's pocket and community cards.
- **HandRankingDisplay.vue**: Visualizes evaluation results.
- **MonteCarloSimulator.vue**: Controls and displays simulation data.
