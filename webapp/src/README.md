# Source Code Directory

This directory contains the source code for the Vue.js Poker Hand Evaluator application.

## Structure

- **components/**: Reusable Vue components (Card, CardSelector, etc.).
- **composables/**: Vue Composition API logic (evaluation, simulation).
- **stores/**: Pinia state management stores.
- **views/**: Top-level page components.
- **assets/**: Static assets like images and global styles.
- **types/**: TypeScript type definitions.
- **utils/**: Helper utility functions.

*Note: This project is currently in the scaffolding phase. Files will be populated according to the [Implementation Plan](../IMPLEMENTATION_PLAN.md).*
